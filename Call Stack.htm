Call Stack

The call stack is a data structure used by JavaScript to keep track of function calls. It operates in a last-in, first-out (LIFO) manner, meaning the last function added to the stack is the first one to be executed.

How It Works

  • When a function is called, it is added to the top of the call stack.
  • The JavaScript engine executes the function at the top of the stack.
  • If that function calls another function, the new function is added to the top of the stack.
  • Once a function completes its execution, it is removed from the stack, and control returns to the function below it.

Example

function firstFunction() {
    console.log("First Function");
    secondFunction();
}
function secondFunction() {
    console.log("Second Function");
    thirdFunction();
}
function thirdFunction() {
    console.log("Third Function");
}
firstFunction();

In this example, when firstFunction is called, it is added to the call stack. It then calls secondFunction, which is added to the stack, and so on. The output will be:

First Function
Second Function
Third Function
